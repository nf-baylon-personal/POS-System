# POS_System
This is a Point of Sales (POS) System application built using Visual Basic .NET 8 and serviced by a MySQL database, capable of basic Create, Read, Update, and Delete (CRUD) operations. Developed in partial fulfillment of the requirements for Computer Programming 4 and Distributed Systems and Databases.

Basic instructions:
1. Import pos_sys.sql into XAMPP or another MySQL service.
2. Open the Releases folder and navigate to find the POS_System.exe file.
3. Run program.

Note: The program will throw an error at start-up if the databases is offline or not set-up properly.
